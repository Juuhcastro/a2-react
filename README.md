## Descrição do Projeto
Este repositório contém o desenvolvimento da atividade avaliativa A2, cujo objetivo foi criar uma página inicial em React seguindo o protótipo disponibilizado no Figma.  
A proposta buscou reforçar a compreensão sobre componentização, organização do código e aplicação de estilização em um projeto real.

A página foi construída com Vite e estruturada com componentes reutilizáveis, respeitando a lógica e a aparência apresentada no modelo.

## Tecnologias Utilizadas
- React  
- Vite  
- JavaScript  
- CSS  
- React Router DOM  

## Estrutura do Projeto
A aplicação foi dividida em componentes para facilitar leitura e manutenção:

- **Header** – Cabeçalho com link para a página inicial  
- **Hero** – Seção principal da Home  
- **Footer** – Rodapé com links e imagem decorativa  
- **PageSection** – Componente reaproveitado nas páginas internas  

Páginas criadas conforme a atividade:
- Home  
- Destinos  
- Ofertas  
- Contato  

## Funcionalidades
- Navegação entre páginas utilizando React Router  
- Layout estruturado em componentes funcionais  
- Páginas adicionais criadas a partir dos links do rodapé  
- Estilização seguindo o protótipo do Figma, incluindo fontes, cores e espaçamentos
